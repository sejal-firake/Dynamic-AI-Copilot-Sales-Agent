🚀 AI Sales Agent – Dynamic Copilot Solution
This project is a complete AI‑powered Sales Agent built using Microsoft Dynamic Copilot, leveraging Topics, Power Automate Flows, Dataverse tables, and custom connectors to automate the entire sales order creation process.
The solution provides a natural, multi‑turn conversational experience that guides a user through authentication, product selection, pricing, address capture, and final order creation — all without writing traditional backend code.

✨ Features
🔐 User Authentication

Validates user identity inside the Copilot conversation
Ensures secure and personalized interactions

🧠 Intelligent Product Assistance

Product suggestions based on user queries
Dynamic selection of product models/variants
Handles clarification seamlessly through multi‑turn dialog

🔢 Quantity & Pricing Calculation

Captures quantity
Uses Power Automate + Dataverse to compute pricing
Returns a detailed price summary back to the user

🏠 Address Capture

Supports both structured and free‑form address input
Validates required fields for order processing

🧾 Order Summary & Order Creation

Displays final confirmation summary
Creates the order in Dataverse through a Power Automate flow
Responds with confirmation message
